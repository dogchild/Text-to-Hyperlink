# 文本转超链接 + 网盘提取码自动填充 (Text To Hyperlink & Cloud Drive Auto-Fill)

一个功能强大的油猴（Tampermonkey）脚本，能够智能识别网页中的纯文本链接并转换为可点击的超链接，同时针对主流网盘提供**提取码自动识别与填充**功能。

## ✨ 核心功能 (Features)

### 1. 🔗 文本转超链接 (Text to Hyperlink)
自动将网页上不可点击的纯文本链接转换为超链接，支持：
- **标准 URL**: `http://`, `https://`, `www.` 开头的网址
- **磁力链接**: `magnet:?xt=...`
- **其他协议**: `tg://`, `ed2k://`, `thunder://` 等
- **IP 地址**: 支持常见的 IPv4 地址
- **非标准网址**: 智能识别无协议头的网址（如 `google.com`），并支持多种顶级域名

### 2. 💾 网盘自动填码 (Auto-Fill Cloud Drive Codes)
当点击网盘链接时，脚本会自动寻找附近的**提取码/密码**，并在打开网盘页面时**自动填充**并尝试提交。

**支持的网盘列表：**
- 百度网盘 (Baidu)
- 阿里云盘 (Aliyun / Alipan)
- 蓝奏云 (Lanzou)
- 123 云盘
- 夸克网盘 (Quark)
- 城通网盘 (Chengtong)
- 天翼云盘 (Tianyi)

### 3. ⚙️ 灵活的设置 (Flexible Settings)
通过 Tampermonkey 扩展菜单，你可以随时控制脚本的行为：
- **全局开关**：一键开启/关闭所有功能。
- **站点黑名单**：在特定网站上禁用链接转换或网盘识别。
- **独立控制**：链接转换和网盘识别功能可以分别开启或关闭。

## 📥 安装 (Installation)
1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Edge, Firefox, Safari 等)。
2. [点击这里安装脚本](#) (链接指向你的 GreasyFork 或 GitHub release)。

## 📖 使用方法 (Usage)
- **自动运行**：脚本安装后会自动在所有网页生效。
- **菜单控制**：点击浏览器右上角的 Tampermonkey 图标，选择本脚本，即可看到如下选项：
    - `✅ 全局：超链接转换已开启` (点击可关闭)
    - `✅ 全局：网盘识别已开启` (点击可关闭)
    - `✅ 本站：超链接转换已开启` (点击可加入黑名单)
    - `✅ 本站：网盘识别已开启` (点击可加入黑名单)
